# Blog

A Lumen package generated by
[laranail/package-scaffolder](https://opensource.simtabi.com/package-scaffolder/) (`--flavor=lumen`).
A `ServiceProvider` binds a repository-backed manager. Register it in Lumen's `bootstrap/app.php`
(`$app->register(...)`), or let [laranail/package-management](https://opensource.simtabi.com/package-management/)
discover it via its `module.json` / `plugin.json`.

## Install

```bash
composer require modules/blog
```

## Usage

```php
use Some\NamespacePath\Blog\Blog;
use Some\NamespacePath\Blog\Repositories\InMemoryPostRepository;

$blog = new Blog(new InMemoryPostRepository);
$post = $blog->create('Hello', 'World');
$blog->all();
```

Swap `InMemoryPostRepository` for your own `PostRepositoryInterface` implementation
(database, file, API — your choice).
