# Blog

A framework-agnostic, vanilla PHP package generated by
[laranail/package-scaffolder](https://opensource.simtabi.com/package-scaffolder/) (`--flavor=vanilla`).
No Illuminate, Eloquent, Blade or service providers — just PSR-4 classes usable in any PHP project.

## Install

```bash
composer require modules/blog
```

## Usage

```php
use Some\NamespacePath\Blog\Blog;
use Some\NamespacePath\Blog\Repositories\InMemoryPostRepository;

$blog = new Blog(new InMemoryPostRepository);
$post = $blog->create('Hello', 'World');
$blog->all();
```

Swap `InMemoryPostRepository` for your own `PostRepositoryInterface` implementation
(database, file, API — your choice).
