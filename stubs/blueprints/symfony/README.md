# Blog

A Symfony package generated by
[laranail/package-scaffolder](https://opensource.simtabi.com/package-scaffolder/) (`--flavor=symfony`).
A no-arg `BlogServiceProvider` self-wires the repository-backed manager and registers into a Symfony
container; [laranail/package-management](https://opensource.simtabi.com/package-management/) can load it
as a runtime module/plugin via its `module.json` / `plugin.json`.

## Install

```bash
composer require modules/blog
```

## Usage

```php
use Some\NamespacePath\Blog\Blog;
use Some\NamespacePath\Blog\Repositories\InMemoryPostRepository;

$blog = new Blog(new InMemoryPostRepository);
$post = $blog->create('Hello', 'World');
$blog->all();
```

Swap `InMemoryPostRepository` for your own `PostRepositoryInterface` implementation
(database, file, API — your choice).
