# Security Policy

## Supported Versions

Security fixes are applied to the latest stable release of
`laranail/package-scaffolder`.

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public
issue. Instead, email the details to **opensource@simtabi.com**.

Please include:

- a description of the vulnerability and its impact;
- steps to reproduce (proof of concept if possible);
- any affected versions you are aware of.

You will receive an acknowledgement as quickly as possible, and we will keep
you informed as the issue is investigated and resolved. Once a fix is released,
we are happy to credit you in the changelog unless you prefer to remain
anonymous.
